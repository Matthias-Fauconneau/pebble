pub static COMMON: InitGuard<Common>

We need to make various bits of data accessible on a system-wide level (all the CPUs access the same data), including from system call and interrupt handlers. I haven't discovered a particularly elegant way of doing that in Rust yet, but this isn't totally awful.

This can be accessed from anywhere in the kernel, and from any CPU, and so access to each member must be controlled by a type such as Mutex or RwLock. This has lower lock contention than locking the entire structure.
