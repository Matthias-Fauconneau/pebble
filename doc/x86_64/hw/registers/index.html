Contains the Ring 0 and Ring 3 code-segment selectors loaded by syscall and sysret, respectively:

IA32_LSTAR

Contains the virtual address of the handler to call upon syscall.

IA32_FMASK

Upon syscall, the value of this MSR is used to mask RFLAGS. Specifically, if a bit is set in this MSR, that bit in RFLAGS is zerod.

IA32_GS_BASE

A virtual address can be stored in this MSR, and acts as the base of the GS segment.
