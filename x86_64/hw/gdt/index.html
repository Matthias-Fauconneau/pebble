CodeSegment
Gdt

A GDT suitable for the kernel to use. It contains two code segments, one for Ring 0 and another for Ring 3. While data segments still exist on x86_64, they are useless, so we instead just load the selector for the null segment into the data segments.

SegmentSelector

An index into the GDT, specifying a particular segment. These are loaded into the segment registers to reference segments.
